# ❌⭕ Tic Tac Toe - Java Edition
A school project — Classic Tic Tac Toe game built in Java.

## 🎮 Description
A simple two-player Tic Tac Toe game implemented in Java as a school project. Players take turns placing their marks on a 3x3 grid, with the goal of getting three in a row. The game features:

* Two-player local gameplay (X vs O)
* Turn-based input system
* Win detection for rows, columns, and diagonals
* Draw detection when the board is full
* Clean console or GUI-based board display

A solid academic exercise for understanding Java OOP and game logic.

---

## 🚀 Getting Started

### Prerequisites

* JDK 8 or later
* Any Java IDE (e.g., IntelliJ, Eclipse, NetBeans) or terminal

### Run the Game

1. Clone the repository:

```bash
git clone https://github.com/IoannisKassios/TicTacToe.git
cd TicTacToe
```

2. Compile and run:

```bash
javac Main.java
java Main
```

3. Players alternate turns entering their move. First to get 3 in a row wins!
